# Data Dictionary

This file explains what each column in `data/orders.csv` means.

| Column | Meaning |
| --- | --- |
| `order_id` | Unique order identifier |
| `order_date` | Date when the order was placed |
| `customer_id` | Unique customer identifier |
| `city` | Customer city |
| `sales_channel` | Marketing or sales channel that brought the order |
| `product_name` | Product purchased by the customer |
| `category` | Product category |
| `quantity` | Number of units purchased |
| `unit_price` | Selling price per unit |
| `unit_cost` | Business cost per unit |
| `discount_pct` | Discount percentage applied to the order |
| `shipping_fee` | Shipping fee paid by the customer |
| `gross_revenue` | Revenue before discount and shipping |
| `discount_amount` | Money subtracted because of discount |
| `net_revenue` | Final revenue after discount and shipping |
| `profit` | Net revenue minus product cost |
